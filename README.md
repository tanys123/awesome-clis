# Awesome Agent CLIs [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated list of command-line interfaces built for AI agents.

CLIs are becoming the new API layer for AI agents. This list tracks tools that are designed to be driven programmatically by agents — non-interactive by default, structured output, and composable.

## Finance & Payments

- [Ramp CLI](https://agents.ramp.com/) - Manage cards, bills, expenses, and approvals from the terminal with an `--agent` mode for structured JSON output.
- [Visa CLI](https://visacli.sh/) - Give AI agents the ability to securely make card payments programmatically.

## Messaging & Communication

- [Kapso CLI](https://github.com/gokapso) - Provision WhatsApp numbers for AI agents via the official Meta WhatsApp Cloud API.
- [Sendblue CLI](https://github.com/sendblue-api) - Give AI agents iMessage phone numbers with a two-step setup.

## Email

- [Resend CLI](https://github.com/resend/resend-cli) - The official CLI for Resend, the developer-focused email sending platform.

## Voice & Audio

- [ElevenLabs CLI](https://github.com/elevenlabs/cli) - Manage voice agents from the terminal. Non-interactive by default so agents and CI/CD can drive it.

## Productivity & Workspace

- [Google Workspace CLI](https://github.com/googleworkspace/cli) - One CLI for Drive, Gmail, Calendar, Sheets, Docs, Chat, Admin, and more — dynamically built from Google Discovery Service.

## Web Scraping & Data

- [Discrawl](https://github.com/steipete/discrawl) - Crawl and query Discord data from the command line with a SQLite backend.
