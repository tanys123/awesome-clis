# Contributing

Thanks for your interest in contributing! Here's how to add a CLI to the list.

## Adding a CLI

1. **One entry per pull request.**
2. Use the format: `- [Name](url) - Description.`
   - Description should be one sentence, ending with a period.
   - Keep it factual and non-promotional.
3. Place entries **alphabetically** within their category.
4. The CLI should be:
   - Designed to work with AI agents (non-interactive, structured output, or agent mode).
   - Open source or publicly available.
   - Maintained (commit in the last 6 months).
   - Have a working README with install instructions.

## Suggesting a new category

Open an issue first to discuss it. Categories should have at least 2 entries.

## Pull request template

```
### CLI Name
- **URL:** https://github.com/...
- **Description:** One sentence.
- **Category:** Existing or proposed category.
- **Why it's agent-friendly:** Brief explanation.
```
